75.6. BKIの例 #

次の一連のコマンドは、それぞれoid型、int4型、text型の3つの列、oid、cola、colbを持ち、OID 420 が付いたtest_tableテーブルを作成し、そして2つの行をテーブルに挿入します。 

create test_table 420 (oid = oid, cola = int4, colb = text)
open test_table
insert ( 421 1 'value 1' )
insert ( 422 2 _null_ )
close test_table
