54.17. pg_publication_tables #

《マッチ度[75.711893]》pg_publication_tablesビューはパブリケーションとそれに含まれるテーブルの間のマッピングに関する情報を提供します。 その元となるカタログpg_publication_relとは異なり、このビューはFOR ALL TABLESとFOR TABLES IN SCHEMAで定義されたパブリケーションを展開するため、そのようなパブリケーションについては対象となる各テーブルについて1行があります。 《機械翻訳》pg_publication_tablesビューは、パブリケーションとそれが含むテーブルの情報間のマッピングに関する情報を提供します。 基礎となるカタログpg_publication_relとは異なり、このビューはpg_publication_relおよびpg_publication_tablesとして定義されたパブリケーションを展開します。 そのため、このようなパブリケーションに対しては、適格な各テーブルに対して1行があります。

表54.17 pg_publication_tablesの列

列 型

説明

pubname name （参照先 pg_publication.pubname）

パブリケーションの名前

schemaname name （参照先 pg_namespace.nspname）

テーブルがあるスキーマの名前

tablename name （参照先 pg_class.relname）

テーブルの名前

attnames name[] （参照先 pg_attribute.attname）

パブリケーションに含まれるテーブル列の名前。 ユーザーがテーブルの列リストを指定しなかった場合に、テーブルのすべての列が含まれます。

rowfilter text

テーブルのパブリケーション必要条件の式
