SPI_palloc

SPI_palloc — 上位エグゼキュータコンテキスト内にメモリを割り当てる

概要

void * SPI_palloc(Size size)

説明

SPI_pallocは上位エグゼキュータコンテキスト内にメモリを割り当てます。

この関数はSPIに接続されている間にのみ使うことができます。 それ以外の場合はエラーを発生させます。

引数

Size size

割り当てる領域のバイト数

戻り値

指定サイズの新しい格納領域へのポインタ
