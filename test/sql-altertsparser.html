ALTER TEXT SEARCH PARSER

ALTER TEXT SEARCH PARSER <refpurpose>change the definition of a text search parser</refpurpose> — テキスト検索パーサの定義を変更する

概要

ALTER TEXT SEARCH PARSER name RENAME TO new_name
ALTER TEXT SEARCH PARSER name SET SCHEMA new_schema

説明

<title>Description</title>

<command>ALTER TEXT SEARCH PARSER</command> changes the definition of a text search parser. Currently, the only supported functionality is to change the parser's name. ALTER TEXT SEARCH PARSERはテキスト検索パーサの定義を変更します。 現時点では、パーサ名称の変更機能のみがサポートされています。

You must be a superuser to use <command>ALTER TEXT SEARCH PARSER</command>. ALTER TEXT SEARCH PARSERを使用するためにはスーパーユーザでなければなりません。

パラメータ

<title>Parameters</title>
name

The name (optionally schema-qualified) of an existing text search parser. 既存のテキスト検索パーサの名称（スキーマ修飾可）です。

new_name

The new name of the text search parser. 新しいテキスト検索パーサの名称です。

new_schema

The new schema for the text search parser. 全文検索パーサの新しいスキーマです。

互換性

<title>Compatibility</title>

There is no <command>ALTER TEXT SEARCH PARSER</command> statement in the SQL standard. 標準SQLにはALTER TEXT SEARCH PARSER文はありません。
