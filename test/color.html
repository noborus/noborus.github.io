<title>Color Support</title>

Most programs in the PostgreSQL package can produce colorized console output. This appendix describes how that is configured. PostgreSQLパッケージのたいていのプログラムは、色付けされたコンソール出力が可能です。 この付録では、色付けがどのように設定されるかを説明します。
