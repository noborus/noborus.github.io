DEALLOCATE

DEALLOCATE <refpurpose>deallocate a prepared statement</refpurpose> — プリペアド文の割り当てを解除する

概要

DEALLOCATE [ PREPARE ] { name | ALL }

説明

<title>Description</title>

<command>DEALLOCATE</command> is used to deallocate a previously prepared SQL statement. If you do not explicitly deallocate a prepared statement, it is deallocated when the session ends. DEALLOCATEを使用して、過去にプリペアドSQL文の割り当てを解除します。 プリペアド文を明示的に割り当て解除しなかった場合、セッションが終了した時に割り当てが解除されます。

For more information on prepared statements, see <xref linkend="sql-prepare"/>. プリペアド文に関する詳細についてはPREPAREを参照してください。

パラメータ

<title>Parameters</title>
PREPARE

This key word is ignored. このキーワードは無視されます。

name

The name of the prepared statement to deallocate. 割り当てを解除する、プリペアド文の名前です。

ALL

Deallocate all prepared statements. プリペアド文の割り当てをすべて解除します

互換性

<title>Compatibility</title>

The SQL standard includes a <command>DEALLOCATE</command> statement, but it is only for use in embedded SQL. DEALLOCATE文は標準SQLにもありますが、埋め込みSQLでの使用のみに用途が限定されています。
