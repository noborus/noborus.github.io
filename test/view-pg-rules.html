54.21. pg_rules

The view <structname>pg_rules</structname> provides access to useful information about query rewrite rules. pg_rulesビューは問い合わせ書き換えルールについての有用な情報へのアクセスを提供します。

表54.21 pg_rulesの列

<title><structname>pg_rules</structname> Columns</title>

Column Type 列 型

Description 説明

schemaname name (references <link linkend="catalog-pg-namespace"><structname>pg_namespace</structname></link>.<structfield>nspname</structfield>) （参照先 pg_namespace.nspname）

Name of schema containing table テーブルがあるスキーマの名前

tablename name (references <link linkend="catalog-pg-class"><structname>pg_class</structname></link>.<structfield>relname</structfield>) （参照先 pg_class.relname）

Name of table the rule is for ルールの対象のテーブル名

rulename name (references <link linkend="catalog-pg-rewrite"><structname>pg_rewrite</structname></link>.<structfield>rulename</structfield>) （参照先 pg_rewrite.rulename）

Name of rule ルール名

definition text

Rule definition (a reconstructed creation command) ルール定義（再構築された生成コマンド）


The <structname>pg_rules</structname> view excludes the <literal>ON SELECT</literal> rules of views and materialized views; those can be seen in <link linkend="view-pg-views"><structname>pg_views</structname></link> and <link linkend="view-pg-matviews"><structname>pg_matviews</structname></link>. pg_rulesビューは、ビューおよびマテリアライズドビューに対するON SELECTルールを除外します。 これらはpg_viewsおよびpg_matviewsにあります。
