<title>Release 13</title>

<title>Release date:</title> リリース日: 2020-09-24

E.2.1. 概要

<title>Overview</title>

<productname>PostgreSQL</productname> 13 contains many new features and enhancements, including: PostgreSQL 13には以下をはじめとする多数の新機能と拡張が含まれます。

  • Space savings and performance gains from de-duplication of B-tree index entries Bツリーインデックス項目の重複除去による省スペース化と性能向上

  • Improved performance for queries that use aggregates or partitioned tables 集約やパーティションテーブルを使う問い合わせの性能改善

  • Better query planning when using extended statistics 拡張統計情報を使ったときのより良い問い合わせの計画作成

  • Parallelized vacuuming of indexes 並列化されたインデックスのバキューム

  • Incremental sorting インクリメンタルソート

The above items and other new features of <productname>PostgreSQL</productname> 13 are explained in more detail in the sections below. PostgreSQL 13の上記の項目とその他の機能は次節でより詳しく説明されます。

<title>Migration to Version 13</title>

A dump/restore using <xref linkend="app-pg-dumpall"/> or use of <xref linkend="pgupgrade"/> or logical replication is required for those wishing to migrate data from any previous release.
