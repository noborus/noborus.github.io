dblink_get_connections

dblink_get_connections <refpurpose>returns the names of all open named dblink connections</refpurpose> — 接続中の名前付きdblink接続すべての名前を返します

概要

dblink_get_connections() returns text[]

説明

<title>Description</title>

<function>dblink_get_connections</function> returns an array of the names of all open named <filename>dblink</filename> connections. dblink_get_connectionsは、接続中の名前付きのdblink接続すべての名前を配列として返します。

戻り値

<title>Return Value</title> <para>Returns a text array of connection names, or NULL if none.</para>

接続名のテキスト型配列を返します。なければNULLです。

例

<title>Examples</title> 
SELECT dblink_get_connections();
