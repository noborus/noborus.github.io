ALTER LANGUAGE

ALTER LANGUAGE <refpurpose>change the definition of a procedural language</refpurpose> — 手続き言語の定義を変更する

概要

ALTER [ PROCEDURAL ] LANGUAGE name RENAME TO new_name
ALTER [ PROCEDURAL ] LANGUAGE name OWNER TO { new_owner | CURRENT_ROLE | CURRENT_USER | SESSION_USER }

説明

<title>Description</title>

<command>ALTER LANGUAGE</command> changes the definition of a procedural language. The only functionality is to rename the language or assign a new owner. You must be superuser or owner of the language to use <command>ALTER LANGUAGE</command>. ALTER LANGUAGEは、手続き言語の定義を変更します。 言語名の変更および言語の所有者の変更のみが可能です。 ALTER LANGUAGEを使用するためにはスーパーユーザまたは言語の所有者でなければなりません。

パラメータ

<title>Parameters</title>
name

Name of a language 言語の名前です。

new_name

The new name of the language 新しい言語の名前です。

new_owner

The new owner of the language 新しい言語の所有者です。

互換性

<title>Compatibility</title>

There is no <command>ALTER LANGUAGE</command> statement in the SQL standard. 標準SQLにはALTER LANGUAGE文はありません。
