17.3. ソースの入手

<title>Getting the Source</title>

The <productname>PostgreSQL</productname> source code for released versions can be obtained from the download section of our website: <ulink url="https://www.postgresql.org/ftp/source/"></ulink>. Download the <filename>postgresql-<replaceable>version</replaceable>.tar.gz</filename> or <filename>postgresql-<replaceable>version</replaceable>.tar.bz2</filename> file you're interested in, then unpack it: 《機械翻訳》リリースされたバージョンのPostgreSQLソースコードは、私たちのウェブサイトのダウンロードセクションhttps://www.postgresql.org/ftp/source/から入手できます。 興味のあるpostgresql-version.tar.gzまたはpostgresql-version.tar.bz2ファイルをダウンロードして解凍します: 

tar xf postgresql-version.tar.bz2

This will create a directory <filename>postgresql-<replaceable>version</replaceable></filename> under the current directory with the <productname>PostgreSQL</productname> sources. Change into that directory for the rest of the installation procedure. 《機械翻訳》これによりpostgresql-versionというディレクトリが、PostgreSQLソースと共に現在のディレクトリの下に作成されます。 残りのインストール手順では、そのディレクトリに移動します。

Alternatively, you can use the Git version control system; see <xref linkend="git"/> for more information. 《機械翻訳》あるいは、Gitバージョン管理システムを使うこともできます。 詳細はI.1を参照してください。
