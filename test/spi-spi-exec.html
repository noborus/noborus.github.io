SPI_exec

SPI_exec — 読み書きコマンドを実行する

概要

int SPI_exec(const char * command, long count)

説明

SPI_execは、常にread_onlyパラメータをfalseとしたSPI_executeと同じです。

引数

const char * command

実行するコマンドを含む文字列。

long count

返される行の最大数。無制限なら0。

戻り値

SPI_executeを参照してください。
