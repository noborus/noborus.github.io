N.1. いつ色が使われるか

<title>When Color is Used</title>

To use colorized output, set the environment variable <envar>PG_COLOR</envar><indexterm><primary>PG_COLOR</primary></indexterm> as follows: 色付けされた出力を使うには、環境変数PG_COLORを以下のように設定します。

  1. If the value is <literal>always</literal>, then color is used. 値がalwaysの場合、色が使われます。

  2. If the value is <literal>auto</literal> and the standard error stream is associated with a terminal device, then color is used. 値がautoで標準エラーストリームがターミナルデバイスに関連付けられている場合、色が使われます。

  3. Otherwise, color is not used. それ以外の場合には色は使われません。
