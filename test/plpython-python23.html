46.10. Python 2対Python 3

<title>Python 2 vs. Python 3</title>

PL/Python supports only Python 3. Past versions of <productname>PostgreSQL</productname> supported Python 2, using the <literal>plpythonu</literal> and <literal>plpython2u</literal> language names. 《機械翻訳》PL/PythonはPython 3のみをサポートしています。 PostgreSQLの以前のバージョンでは、plpythonuとplpython2uという言語名を使用してPython 2がサポートされていました。
