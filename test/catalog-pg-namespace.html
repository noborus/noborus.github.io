53.32. pg_namespace #

pg_namespaceカタログは名前空間を保存します。 名前空間はSQLスキーマの裏にある構造です。 それぞれの名前空間は、リレーション、型などの集合を、名前が競合することなく、個別に持ちます。

表53.32 pg_namespaceの列

列 型

説明

oid oid

行識別子

nspname name

名前空間の名前

nspowner oid （参照先 pg_authid.oid）

名前空間の所有者

nspacl aclitem[]

アクセス権限。 詳細は5.7を参照してください
