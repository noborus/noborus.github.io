E.347. リリース7.0.3

<title>Release 7.0.3</title>

<title>Release date:</title> リリース日: 2000-11-11

This has a variety of fixes from 7.0.2. このリリースは7.0.2の不具合を改修したものです。

E.347.1. バージョン7.0.3への移行

<title>Migration to Version 7.0.3</title>

A dump/restore is <emphasis>not</emphasis> required for those running 7.0.*. 7.0.*からの移行の場合はダンプ/リストアは必要ありません。
