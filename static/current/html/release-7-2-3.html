E.339. リリース7.2.3

<title>Release 7.2.3</title>

<title>Release date:</title> リリース日: 2002-10-01

This release contains a variety of fixes for version 7.2.2, including fixes to prevent possible data loss. このリリースはバージョン7.2.2の不具合を改修したものです。データ損失の可能性を防止するための改修も含まれています。

E.339.1. バージョン7.2.3への移行

<title>Migration to Version 7.2.3</title>

A dump/restore is <emphasis>not</emphasis> required for those running version 7.2.*. 7.2.*からの移行の場合はダンプ/リストアは必要ありません。

E.339.2. 変更点

<title>Changes</title>

  • Prevent possible compressed transaction log loss (Tom)

  • Prevent non-superuser from increasing most recent vacuum info (Tom)

  • Handle pre-1970 date values in newer versions of glibc (Tom)

  • Fix possible hang during server shutdown

  • Prevent spinlock hangs on SMP PPC machines (Tomoyuki Niijima)

  • Fix pg_dump to properly dump FULL JOIN USING (Tom)
