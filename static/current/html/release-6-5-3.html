E.351. リリース6.5.3

<title>Release 6.5.3</title>

<title>Release date:</title> リリース日: 1999-10-13

This is basically a cleanup release for 6.5.2. We have added a new <application>PgAccess</application> that was missing in 6.5.2, and installed an NT-specific fix. このリリースは基本的には 6.5.2 を整理したものです。 6.5.2 でなくなったPgAccess を新しく追加し、また、NT に特化した改修を組み込みました。

E.351.1. バージョン6.5.3への移行

<title>Migration to Version 6.5.3</title>

A dump/restore is <emphasis>not</emphasis> required for those running 6.5.*. 6.5.*からの移行の場合はダンプ/リストアは必要ありません。

E.351.2. 変更点

<title>Changes</title> 

Updated version of pgaccess 0.98
NT-specific patch
Fix dumping rules on inherited tables
