E.7. リリース10.2

<title>Release 10.2</title>

<title>Release date:</title> リリース日: 2018-02-08

This release contains a variety of fixes from 10.1. For information about new features in major release 10, see <xref linkend="release-10"/>. このリリースは10.1に対し、様々な不具合を修正したものです。 10メジャーリリースにおける新機能については、E.9を参照してください。

E.7.1. バージョン10.2への移行

<title>Migration to Version 10.2</title>

A dump/restore is not required for those running 10.X. 10.Xからの移行ではダンプ/リストアは不要です。

However, if you use <filename>contrib/cube</filename>'s <literal>~&gt;</literal> operator, see the entry below about that. しかしながら、contrib/cubeの~>演算子を使っているなら、これに関する下記の項目を参照してください。

Also, if you are upgrading from a version earlier than 10.1, see <xref linkend="release-10-1"/>. また、10.1よりも前のバージョンからアップグレードする場合、E.8を参照してください。
