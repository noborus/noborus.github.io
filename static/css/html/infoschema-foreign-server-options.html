37.27. foreign_server_options

foreign_server_optionsビューには、現在のデータベース内の外部サーバ用に定義されたすべてのオプションが含まれます。 現在のユーザが（所有者である、または何らかの権限を持つことにより）アクセス可能な外部サーバのみが表示されます。

表37.25 foreign_server_optionsの列

名前データ型説明
foreign_server_catalogsql_identifier外部サーバを定義したデータベースの名前です（常に現在のデータベースです）。
foreign_server_namesql_identifier外部サーバの名前
option_namesql_identifierオプションの名前
option_valuecharacter_dataオプションの値
