37.12. column_domain_usage

The view <literal>column_domain_usage</literal> identifies all columns (of a table or a view) that make use of some domain defined in the current database and owned by a currently enabled role. column_domain_usageビューは、現在のデータベース内で定義され、現在有効なロールが所有するあるドメインを使用する（テーブルもしくはビューの）全ての列を識別します。
