E.36. リリース9.5.1

<title>Release 9.5.1</title>

<title>Release date:</title> リリース日: 2016-02-11

This release contains a variety of fixes from 9.5.0. For information about new features in the 9.5 major release, see <xref linkend="release-9-5"/>. このリリースは9.5.0に対し、各種不具合を修正したものです。 9.5メジャーリリースにおける新機能については、E.37 を参照してください。

E.36.1. バージョン9.5.1への移行

<title>Migration to Version 9.5.1</title>

A dump/restore is not required for those running 9.5.X. 9.5.Xからの移行ではダンプ/リストアは不要です。
