52.82. pg_rules

The view <structname>pg_rules</structname> provides access to useful information about query rewrite rules. pg_rulesビューは問い合わせ書き換えルールについての有用な情報へのアクセスを提供します。

The <structname>pg_rules</structname> view excludes the <literal>ON SELECT</literal> rules of views and materialized views; those can be seen in <structname>pg_views</structname> and <structname>pg_matviews</structname>. pg_rulesビューは、ビューおよびマテリアライズドビューに対するON SELECTルールを除外します。 これらはpg_viewsおよびpg_matviewsにあります。
