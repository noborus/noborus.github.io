E.53. リリース9.4.5

<title>Release 9.4.5</title>

<title>Release date:</title> リリース日: 2015-10-08

This release contains a variety of fixes from 9.4.4. For information about new features in the 9.4 major release, see <xref linkend="release-9-4"/>. このリリースは9.4.4に対し、各種不具合を修正したものです。 9.4メジャーリリースにおける新機能については、E.58を参照してください。

E.53.1. バージョン9.4.5への移行

<title>Migration to Version 9.4.5</title>

A dump/restore is not required for those running 9.4.X. 9.4.Xからの移行ではダンプ/リストアは不要です。

However, if you are upgrading from a version earlier than 9.4.4, see <xref linkend="release-9-4-4"/>. しかしながら、9.4.4よりも前のバージョンからアップグレードする場合は、E.54を参照して下さい。
