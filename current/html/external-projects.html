<title>External Projects</title>

<productname>PostgreSQL</productname> is a complex software project, and managing the project is difficult. We have found that many enhancements to <productname>PostgreSQL</productname> can be more efficiently developed separately from the core project. PostgreSQLは複雑なソフトウェアプロジェクトであり、そのプロジェクト管理は困難です。 PostgreSQLに対する拡張をコアプロジェクトと分離して開発する方がより効率的であることがわかりました。
