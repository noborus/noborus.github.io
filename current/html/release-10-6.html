E.3. リリース 10.6

<title>Release 10.6</title>

<title>Release date:</title> リリース日: 2018-11-08

This release contains a variety of fixes from 10.5. For information about new features in major release 10, see <xref linkend="release-10"/>. このリリースは10.5に対し、様々な不具合を修正したものです。 10メジャーリリースにおける新機能については、E.9を参照してください。

E.3.1. バージョン10.6への移行

<title>Migration to Version 10.6</title>

A dump/restore is not required for those running 10.X. 10.Xからの移行ではダンプ/リストアは不要です。

However, if you use the <filename>pg_stat_statements</filename> extension, see the changelog entry below about that. しかしながら、pg_stat_statements拡張を使っているなら、下記のそれに関する変更履歴項目を確認してください。

Also, if you are upgrading from a version earlier than 10.4, see <xref linkend="release-10-4"/>. また、10.4よりも前のバージョンからアップグレードする場合には、E.5を参照してください。
