52.34. pg_operator

The catalog <structname>pg_operator</structname> stores information about operators. See <xref linkend="sql-createoperator"/> and <xref linkend="xoper"/> for more information. pg_operatorカタログは演算子の情報を保存します。 CREATE OPERATORと38.13を参照してください。

表52.34 pg_operatorの列

oidoid 行識別子（隠し属性です。明示的に選択しなければなりません）
