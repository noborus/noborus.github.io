E.306. リリース7.4.5

<title>Release 7.4.5</title>

<title>Release date:</title> リリース日: 2004-08-18

This release contains one serious bug fix over 7.4.4. For information about new features in the 7.4 major release, see <xref linkend="release-7-4"/>. このリリースは7.4.4の深刻な不具合を修正したものです。 7.4メジャーリリースにおける新機能についてはE.311を参照してください。

E.306.1. 7.4.4バージョンへの移行

<title>Migration to Version 7.4.5</title>

A dump/restore is not required for those running 7.4.X. 7.4.Xからの移行の場合は ダンプ/リストアは必要ありません。

E.306.2. 変更点

<title>Changes</title>
<listitem><para>Repair possible crash during concurrent B-tree index insertions</para>

  • B-treeインデックスの同時挿入時にクラッシュする可能性を修正

    This patch fixes a rare case in which concurrent insertions into a B-tree index could result in a server panic. No permanent damage would result, but it's still worth a re-release. The bug does not exist in pre-7.4 releases. このパッチは、B-treeインデックスへの同時挿入時にサーバパニックを引き起こす稀な問題を修正します。 永続的な損傷にはなりませんが、リリースし直すべきであると判断しました。 7.4より前のリリースには、この不具合はありません。
