<title>Documentation</title>

<productname>PostgreSQL</productname> has four primary documentation formats: PostgreSQLの文書には4つの主要なフォーマットがあります。

  • Plain text, for pre-installation information 平文：インストール前の情報

  • <acronym>HTML</acronym>, for on-line browsing and reference HTML：オンラインでの閲覧と参照

  • PDF, for printing PDF：印刷用

  • man pages, for quick reference. マニュアルページ：すぐ知りたい時

Additionally, a number of plain-text <filename>README</filename> files can be found throughout the <productname>PostgreSQL</productname> source tree, documenting various implementation issues. さらに、PostgreSQLソースツリー全体にわたり、様々な実装に関する問題を記述した平文のREADMEファイルが数多くあります。

<acronym>HTML</acronym> documentation and man pages are part of a standard distribution and are installed by default. PDF format documentation is available separately for download. HTML文書とマニュアルページは標準の配布物の一部でデフォルトでインストールされます。 PDF形式の文書は別途ダウンロードすることで入手できます。
