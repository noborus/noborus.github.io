E.245. リリース8.1.8

<title>Release 8.1.8</title>

<title>Release date:</title> リリース日: 2007-02-07

This release contains one fix from 8.1.7. For information about new features in the 8.1 major release, see <xref linkend="release-8-1"/>. このリリースには、8.1.7に対する1つの修正が含まれています。 8.1メジャーリリースにおける新機能についてはE.253を参照してください。

E.245.1. バージョン8.1.8への移行

<title>Migration to Version 8.1.8</title>

A dump/restore is not required for those running 8.1.X. However, if you are upgrading from a version earlier than 8.1.2, see <xref linkend="release-8-1-2"/>. 8.1.Xからの移行ではダンプ/リストアは不要です。 しかしながら、8.1.2より前のバージョンからアップグレードする場合は、E.251を参照してください。

E.245.2. 変更点

<title>Changes</title>

  • Remove overly-restrictive check for type length in constraints and functional indexes(Tom) 制約および関数インデックスにおける型の長さに対する重複/限定検査を削除しました。(Tom)
