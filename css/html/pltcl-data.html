44.3. PL/Tclにおけるデータの値

PL/Tcl関数コードに与えられる引数の値は、単に、テキスト形式（SELECT文によりそれを表示した場合と同じ形式）に変換された入力引数です。 逆に、returnコマンドとreturn_nextコマンドは、その関数宣言における戻り値の型、あるいは複合型の戻り値型の入力書式として受け付けることができる、任意の文字列を受け付けます。
