37.3. information_schema_catalog_name

information_schema_catalog_nameは、常に現在のデータベース名（SQL用語では現在のカタログ）を持つ1行1列からなるテーブルです。

表37.1 information_schema_catalog_nameの列

名前データ型説明
catalog_namesql_identifierこの情報スキーマを持つデータベースの名前です。
